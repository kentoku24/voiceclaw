# セキュリティ方針（暫定）

- 更新日: 2026-03-01

## 重要

- **Publicリポジトリにトークン/鍵/APIキー等の秘密情報を置かない**
- **ブラウザにDiscord Botトークンを入力させる設計はNG（MVPでも原則避ける）**
  - クライアントは悪意ある拡張・XSS・ログ等で漏洩しうる

## MVPでの推奨

- Discord Botトークンはサーバ側（中継API）で保持
- クライアント（Web）は
  - 録音開始/終了（UI）
  - STT（ブラウザ内蔵でもOK）
  - 中継APIへテキスト送信
  のみ

## インシデント時

- 誤ってトークンを公開コミットした場合:
  1) 即座に削除
  2) **履歴からの削除（history rewrite）**
  3) トークンの **失効/再発行**（Discord側）
  4) 影響範囲確認（権限/スコープ/ログ）
